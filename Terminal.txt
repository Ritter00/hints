Запуск:
Win+r	или cmd # cmd можно набрать в адресной строке и терминал откроется на этой папке

Команды:
dir 	# просмотр - что в папке
dir/A	# просмотр скрытых файлов
dir/AH	# вывод только скрытых файлов
del text.txt	# удалить файл text.txt
rd	# удалить папку
rd/S	# удалить папку с содержимым /q	# тоже, но без подтверждения

cd <имя папки>	# ПЕРЕХОД в папку ex. - cd User (TAB - автозаполнение после первых символов)
cd .. 	#  на один уровень вверх (cd ../..   	на два уровня)
С:	# перейти на диск С (D: -  на диск D)  и аналогично
mkdir projects	# создать папку "projects"
